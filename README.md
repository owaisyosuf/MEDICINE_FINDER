# MEDICINE_FINDER
Project Title: Karachi Medicine Finder
Project Description:
Karachi Medicine Finder is a web-based app developed to solve a real problem faced by people in Karachi — difficulty finding medicines, comparing prices, and locating nearby pharmacies. The app was built using HTML, CSS, and JavaScript with the assistance of AI tools including Claude and ChatGPT.
The app allows users to search any medicine by name and instantly get complete information including what illnesses or conditions that medicine treats, dosage guidelines, and side effect warnings. It also shows all medicines that share the same active formula so users can find cheaper generic alternatives. A price comparison table displays all brand and generic options side by side, showing how much money the user can save by switching to a generic version.
The app also includes a pharmacy finder feature that shows nearby pharmacies in Karachi with availability status. If the medicine is not available at nearby pharmacies, the app automatically searches across all of Karachi and displays which pharmacies have it in stock.
Features:

Medicine search by name, generic name, or illness
Illness and uses information with dosage guide and side effects
Same-formula alternatives (generic medicines)
Price comparison with savings calculator
Nearest pharmacy finder with availability
Karachi-wide pharmacy search as fallback
Live price indicators

AI Tools Used: Claude AI, ChatGPT
Real-World Problem Solved: This app helps Karachi residents find affordable medicine alternatives, understand what a medicine is used for, and locate the nearest pharmacy that has it in stock — saving time, money, and effort.
Note: This app is developed for learning and educational purposes only. Medicine prices and pharmacy availability data are dummy/sample data. Medicine names and medical information are real. Always consult a licensed doctor or pharmacist before taking any medicine.
